# bot-leyu-resource

## 简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内的页面内容不针对任何特定域名或网站，仅作为通用资源存储与展示使用。

## 目录说明

```
/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，命名清晰，便于查找。
- `assets/`：页面所需的样式、脚本、多媒体等资源文件，按需组织子目录。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容保持中立，不包含指向特定网站或服务的链接。
- 归档目的为长期保存与公开访问，页面内容可能不定期更新或调整。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正内容。
- 如需引用或使用仓库内页面，请自行评估适用性。
- 欢迎通过 Issues 提交建议或报告问题，但请注意本仓库不针对特定项目或网站提供支持。

## 使用方式

可直接通过 GitHub Pages 或本地浏览器打开 `pages/` 目录下的 HTML 文件查看内容。具体访问方式请参考 GitHub 相关文档。
